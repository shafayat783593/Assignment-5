Question-1: What are the different ways to select an element in the DOM

There are the many different ways to select an element in the DOM , as like getElementsByTagName(), getElementsByClassName(), getElementById(), querySelector(), querySelectorAll()

Question-2:What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Get html content and tags. innerText: Get visible text only. textContent: Get all text content and whitespace.

Question-3:What is event delegation in the DOM?

Event delegation is using one event listener on a parent to handle events for all its children.

Question-4: What is event bubbling in the DOM?

Event bubbling is like a chain reaction. When something happens on a child element , that event bubbles up to the parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

I can create, add, and remove elements from the DOM using various methods As like Create element: Use document.createElement(), Add element: Use appendChild(), append(), Remove element: Use remove().
